Ways you can help

Start an issue on GitHub

This site is hosted on GitHub. You can report any glitch or your suggestions my starting an issue on the github repo page. This is the esiest and fastest method, but first you have to have a GitHub account first. Link to GitHub repo

Contribute using pull request on GitHub

If you want to add or ammend some part of the website you can make pull requests to the site repo. This is again the easiest method to help, because then I'll merge that pull request to the repo and you've contributed to this wiki. But again, you must have the knowledge how to work on GitHub and how to make pull requests.

If you want to contribute alternatively by directly contacting me, you can do that also. You can contact me.

Help me financially

I am working with a single monitor with low resolution, if I get an extra monitor work would be alot easier for me. So, if you can afford you can always patron this site and show your love to literature.
